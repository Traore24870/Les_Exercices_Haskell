HC2T7 - Tâche 7 : Expressions booléennes
---

### 1. `True` en utilisant `&&` (ET logique)

L'opérateur `&&` ne retourne `True` que si les deux conditions sont vraies.

```haskell
(5 > 3) && (10 == 10)
-- Résultat : True

```

### 2. `False` en utilisant `||` (OU logique)

L'opérateur `||` ne retourne `False` que si les deux conditions sont fausses.

```haskell
(1 > 10) || (2 == 3)
-- Résultat : False

```

### 3. `True` en utilisant `not` (Négation)

L'opérateur `not` inverse la valeur booléenne. Pour obtenir `True`, il faut nier une valeur fausse.

```haskell
not (4 < 2)
-- Résultat : True

```

### 4. Une comparaison qui retourne `False`

Il suffit d'écrire une inégalité ou une égalité mathématiquement fausse.

```haskell
2025 < 1999
-- Résultat : False

```
